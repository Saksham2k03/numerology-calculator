This application converts traditional calculation logic into an interactive web interface:

1.  **Basic Profile:** Instantly calculates your **Birth Number** (Moolank) and **Destiny Number** (Bhagyank).
2.  **Lo Shu Grid:** Generates the standard 3x3 Numerology matrix grid based on your specific birth date frequencies.

5.  **Daily Forecast:** Generates daily numerology predictions based on planetary lords for any given month.
